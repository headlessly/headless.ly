Track developers who sign up, startups using the platform, investor conversations, partner deals.

## Contact

People — developers, founders, VCs, partners.

```typescript
import { Noun } from 'digital-objects'

export const Contact = Noun('Contact', {
  name: 'string!',
  email: 'string?#',
  phone: 'string?',
  title: 'string?',
  stage: 'Lead | Qualified | Customer | Churned | Partner',
  source: 'string?',
  company: '-> Company.contacts',
  deals: '<- Deal.contact[]',
  tickets: '<- Ticket.contact[]',
  customer: '<- Customer.contact',
  messages: '<- Message.contact[]',
  qualify:  'Qualified',
  capture:  'Captured',
  assign:   'Assigned',
  merge:    'Merged',
  enrich:   'Enriched',
})
```

| Verb | Event | Description |
|------|-------|-------------|
| `qualify` | `Qualified` | Move from Lead to Qualified |
| `capture` | `Captured` | Capture from form or import |
| `assign` | `Assigned` | Assign to a team member |
| `merge` | `Merged` | Merge duplicate contacts |
| `enrich` | `Enriched` | Enrich with external data |
