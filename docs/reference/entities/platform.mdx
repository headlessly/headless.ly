## Agent

AI agents with the full autonomous-agents interface. Agents can do work, ask questions, make decisions, approve requests, notify humans, delegate to other agents, escalate when stuck, and learn from outcomes.

```typescript
export const Agent = Noun('Agent', {
  name: 'string!',
  role: 'string!',
  description: 'text?',
  mode: 'Autonomous | Supervised | Manual',
  status: 'Idle | Thinking | Acting | Waiting | Error',
  model: 'string?',
  system: 'text?',
  skills: 'json?',
  tools: 'json?',

  // Core verbs
  do:       'Done',
  ask:      'Asked',
  decide:   'Decided',
  approve:  'Approved',
  notify:   'Notified',
  delegate: 'Delegated',
  escalate: 'Escalated',

  // Learning
  learn:   'Learned',
  reflect: 'Reflected',

  // Lifecycle
  deploy: 'Deployed',
  pause:  'Paused',
  stop:   'Stopped',
})
```

### Core Verbs

| Verb | Event | Description |
|------|-------|-------------|
| `do` | `Done` | Execute a task autonomously |
| `ask` | `Asked` | Ask a question (to human or another agent) |
| `decide` | `Decided` | Choose between options with reasoning |
| `approve` | `Approved` | Approve or reject a request |
| `notify` | `Notified` | Send a notification via any channel |
| `delegate` | `Delegated` | Hand off work to another agent |
| `escalate` | `Escalated` | Escalate to a higher-tier agent or human |

### Learning Verbs

| Verb | Event | Description |
|------|-------|-------------|
| `learn` | `Learned` | Record an insight from an interaction |
| `reflect` | `Reflected` | Self-analyze past actions for improvement |

### Lifecycle Verbs

| Verb | Event | Description |
|------|-------|-------------|
| `deploy` | `Deployed` | Activate the agent |
| `pause` | `Paused` | Temporarily suspend |
| `stop` | `Stopped` | Shut down the agent |

### Modes

| Mode | Description |
|------|-------------|
| `Autonomous` | Agent acts independently, escalates only when stuck |
| `Supervised` | Agent proposes actions, human approves before execution |
| `Manual` | Agent provides recommendations, human executes |

### Durable Object Backing

Each Agent is backed by a Durable Object with persistent:
- **Memory** — long-term storage with importance scoring and semantic recall
- **Conversations** — message history and context across sessions
- **Goals** — objectives with progress tracking and sub-goals
- **Learning** — insights categorized as behavior, knowledge, skill, preference, or error
- **Actions** — registered capabilities with execution tracking and feedback
